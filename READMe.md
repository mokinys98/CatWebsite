# Kačiukų svetainę

## Aprašymas

Nėra

## Reikalavimai

- Node.js
- npm

## Diegimas

1. Klonuokite repozitoriją:
   
   ````
   git clone https://github.com/mokinys98/CatWebsite.git
   ````
2. Eikite į projekto katalogą:
   
   ````
   cd CatWebsite
   ````
3. Įdiekite priklausomybes:
   
   ````
   npm install
   ````
4. Paleiskite serveri
   
   ````
   node src/server.js
   ````
